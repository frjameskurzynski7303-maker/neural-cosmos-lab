# 🌌 Neural Cosmos Lab

A browser-based universe simulation with:
- star formation
- supernova events
- expansion physics
- AI tutor (demo mode)

## 🚀 Deploy to GitHub Pages

1. Create a GitHub repo called:
   neural-cosmos-lab

2. Upload these files:
   - index.html
   - style.css
   - app.js

3. Go to:
   Settings → Pages

4. Set:
   Branch: main / root

5. Click Save

Your site will be live at:
https://frjameskurzynski7303-maker.github.io/neural-cosmos-lab/
